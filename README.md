** QR Code Generator**

This project is a simple and efficient QR Code Generator built using HTML, CSS, and JavaScript. It allows users to instantly convert any text or URL into a QR code with just a single click.

**Features**

**Instant QR Code Generation**
=>Enter any URL or text into the input box, and the app dynamically generates a QR code in real time.

**Easy Sharing**
=>The generated QR code can be shared effortlessly using the built-in Share button.

**Clean & Responsive UI**
=>Designed with a simple, modern layout that works smoothly across devices.



**Technologies Used**

=>HTML5

=>CSS3

=>JavaScript
